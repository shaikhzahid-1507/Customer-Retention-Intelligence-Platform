# 📖 Project Overview

Customer churn is one of the biggest challenges faced by subscription-based businesses. Retaining an existing customer is often significantly more cost-effective than acquiring a new one.

This project uses Machine Learning to predict whether a customer is likely to churn based on demographic information, subscribed services, contract details, and billing history.

The platform goes beyond prediction by providing:

- Executive dashboards
- Business insights
- Revenue-at-risk analysis
- Customer retention recommendations
- Downloadable PDF reports
- Prediction history stored in SQLite

The objective is to enable business teams to identify high-risk customers and take proactive retention actions.
